垂直居中

在css中，vertical-align只能运用在行内元素或者替换元素中，并且该属性不能够被继承。vertical-align只接受8个关键字、一个百分数值或一个长度。其中关键字包括：baseline(默认)、sub、supper、bottom、text-bottom、middle、top、text-top

  1. vertical-align 不影响块级元素中的内容对齐。不过，可以用它来影响表格单元格中的垂直对齐。

    效果:
    姓名 年龄
    Jack 30
    Tom 30
    代码:
    			.table_vl .table-cell
				{
					border:solid 1px #ccc;
					height:80px;
					vertical-align: top;
				}
				<table class="table_vl">
						<thead>
							<tr>
								<th>姓名</th>
								<th>年龄</th>
							</tr>
						</thead>
						<tbody>
							<tr>
								<td class="table-cell">Jack</td>
								<td class="table-cell">30</td>
							</tr>
							<tr>
								<td class="table-cell">Tom</td>
								<td class="table-cell">30</td>
							</tr>
						</tbody>
					</table>
  2. 看看top、text-top、bottom、text-bottom区别.从下面的效果图可以看出。top是指元素的顶部(行高而不是内容高度)移动到行高的位置，而text-top是指元素的顶部移动到父元素文本高度的位置。同理，bottom和text-bottom也是一样。
    效果:

    top text-top bottom text-bottom middle

    代码:
    		.bigLineHeight {
			background:#ff4400;
			color:#fff;
			line-height:300px;
			font-size: 20px;
			border: solid 1px #ccc;
		}
		.bigLineHeightSpan{
			background:#000;
		}
		.vertical-top {
			vertical-align: top;
		}
		.vertical-text-top {
			vertical-align: text-top;
		}
		.vertical-bottom {
			vertical-align: bottom;
		}
		.vertical-text-bottom {
			vertical-align: text-bottom;
		}
		.vertical-middle{
			vertical-align: middle;
		}
