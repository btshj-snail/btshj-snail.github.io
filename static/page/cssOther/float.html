对于浮动元素有几点要注意，首先会以某种模式将浮动元素从文档的正常流中删除，不过他们还是会影响布局。比如浮动元素设置了margin，那么他也会影响周围的其他元素。如果确实要浮动一个非 替换元素，那么一定要设置元素的width，否则浏览器会将其width设置为0

float
值 left | right | none
初始值 none
应用于 所有元素
继承性 无
计算值 根据指定确定

  1. 浮动元素会生成一个块级框，所以将元素添加display:block 没有意义。

  2. 浮动元素的左（右）外边距不能超过其包含块的左（右）内边界

    效果:
    这个框，有30px的外边距，50px的内边距 该元素浮动

  3. 浮动元素的左（右）外边界必须是源文档中之前出现的左浮动元素（右浮动元素）的右（左）边界，除非后出现的浮动元素的顶端在先出现的浮动元素的底端下面。 这个规则的好处在于使用浮动时，不必担心浮动元素的相互覆盖。

    效果:
    这个框，有30px的外边距，50px的内边距 该元素浮动
