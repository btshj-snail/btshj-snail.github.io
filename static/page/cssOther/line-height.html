ine-height

  1. 百分比、ex、em等相对单位，都是相对当前元素的font-size来的。比如font-size为10px，那么line-height：1.5em ，其实相当于是line-height:15px

    效果:

    this line-height is 1.5em

    this font is 10,line-height is 150%

    this line-height is 0.33in

  2. 当一个块级元素从另一个元素继承line-height时。line-height值要从父元素计算，而不是在子元素上计算。如果line-height值为数字的时候，这时叫做缩放因子，此时传递到子元素的是缩放因子而不是计算好的值。因此此时line-height是根据本身元素的font-size进行计算的。
    效果:
    在父元素上设置了line-height的缩放因子，并且font-size为10px
