文本描述

其实在文本上就是各种线

text-decoration
值 none(原始) |　[ underline（下划线） | overline（顶端画线） | line-through（贯穿线） | blink（文本闪烁,基本都不支持该属性） ］| inherit
初始值 none
应用于 所有元素
继承性 无
计算值 根据指定确定
  1. 效果:

    this is underline

    this is overline

    this is line-through

    this is line-blink

    代码:
    			.text_decoration_underline{
                text-decoration:underline;
                margin-top:20px;
            }
            .text_decoration_overline{
                text-decoration:overline;
                margin-top:20px;
            }
            .text_decoration_line_through{
                text-decoration:line-through;
                margin-top:20px;
            }
            .text_decoration_blink{
                text-decoration:blink;
                margin-top:20px;
            }
             <p class="text_decoration_underline">this is underline</p>
            <p class="text_decoration_overline">this is overline</p>
            <p class="text_decoration_line_through">this is line-through</p>
            <p class="text_decoration_blink">this is line-blink</p>

  2. 可以在一个规则中结合多个装饰。但是如果不同的装饰都与一个元素匹配，生出规则的值将会完全取代另一个值

    效果:

    这个规则是含有underline 和line-through

    这个规则只有overline

    有两个class，分别是text_decoration_underline_through和text_decoration_overline，text_decoration_overline被匹配，因此只有overline效果
