  1. 元素的边框就是指围绕元素内容和内边距的一条或多条线。因此，元素的背景会在外边框边界处停止，因为背景不会延伸到外边距以内，而边框就在外边距内部。每个边框有3个方面：宽度、粗细和颜色。 边框默认宽度为medium，这个值没有明确定义，不过通常是2像素。尽管如此，也不一定能看到边框，因为边框样式一般都是none。默认的边框颜色为元素本身的前景色，如果没有为边框申明颜色 则其颜色为文本的颜色，如果元素没有文本，则其边框颜色就是父元素的文本颜色（因为color是可以继承的）。

    效果:

    边框颜色为当前元素的文本颜色

    当前元素没有文本，则边框颜色为父元素的文本颜色

    代码:
    			   .border_parent{
                  color:#ff4400;
                }
                .border_p{
                    border:solid 1px ;
                   width:200px;
                    margin:20px auto;
                    color:#000;
                    height:60px;
                }
                .border_p2{
                    border:solid 1px ;
                    width:200px;
                    margin:auto;
                    height:60px;
                }
				<div>
                    <div class="border_parent">
                        <p class="border_p">边框颜色为当前元素的文本颜色</p>
                    </div>
                    <div class="border_parent">
                        <p class="border_p2">当前元素没有文本，则边框颜色为父元素的文本颜色</p>
                    </div>

                </div>

  2. 边框是在背景上的。在发布css2的时候，背景只延伸到内边距。后来在css2.1的时候，指出元素的背景内容包含内边距、边框和内容区。在“可间断”边框下，则在边框可见部分能看到背景色

    效果:
    如果您能在虚线边框的可见部分看到黑色，则表明你的浏览器支持css2.1 ,如果虚线边框的可见部分不是黑色，则表明您的浏览器只支持css2.0
